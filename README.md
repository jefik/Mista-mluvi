# Místa, která mluví

Mapa ČR která umožňuje uživatelům anonymně přidat bod(pin) na jakékoliv místo (souřadnici) se zprávou.

## Myšlenka projektu

Projekt umožňuje lidem zanechat anonymní vzkaz na místě, které pro ně má význam. Ostatní uživatelé si tyto vzkazy mohou přečíst přímo na mapě.

## Tehnologie

Backend
* Express.js

Frontend
* React
